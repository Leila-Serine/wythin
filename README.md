# Wythin

Wythin is a modern personal growth and productivity platform designed to help users organize their goals, tasks, habits, and focus sessions.

## Features

* Goal management
* Task organization
* Habit tracking
* Focus timer
* Daily inspiration
* Personal progress overview

## Technologies

* Next.js
* React
* TypeScript
* Tailwind CSS
* Lucide React

## Installation

Install the dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open the application at:

```text
http://localhost:3000
```
